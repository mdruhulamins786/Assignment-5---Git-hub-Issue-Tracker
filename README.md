1, What is the difference between var, let, and const?
   Ans:- var is the old way to declare variables (before ES6).

    * Key features:
    * Function scoped (not block scoped)
    * Can be redeclared
    * Can be updated

2️, What is the spread operator (...)?
    Ans:- The spread operator (...) in JavaScript is used to expand or unpack elements from an array, object, or iterable into individual elements.

3️, What is the difference between map(), filter(), and forEach()?
   Ans:- three show up everywhere in JavaScript, especially when working with arrays.

4️, What is an arrow function?
   Ans:- arrow function is a shorter way to write functions in JavaScript, introduced in ES6 (2015). It uses the => arrow syntax.

5️, What are template literals?
   Ans:- Template literals are a modern JavaScript feature that lets you create strings more easily and dynamically using backticks ( ) instead of quotes.

   *They were introduced in ES6.
